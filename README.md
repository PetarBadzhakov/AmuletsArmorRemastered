Amulets & Armor: Remastered
==

A fan project made to bring the game to modern machines and operating systems.

# Goals:

Thorough refactor, debug, optimisation.

Migrate to VS2022, then CMake.

Migrate to SDL3.

Resolution slider with integer upscaling.
